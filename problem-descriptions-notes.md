🟢 Problem 1 (Easy)
Title: Student Result Analyzer

একটা array দেওয়া আছে।

const marks = [85, 92, 78, 96, 88, 74];

কোনো built-in method যেমন Math.max() বা Math.min() ব্যবহার করা যাবে না।

নিচের তথ্যগুলো বের করো:

Highest Mark
Lowest Mark
Total Marks
Average Marks (২ দশমিক পর্যন্ত)
কতজন Student Pass করেছে (Pass Mark = 40)
Requirements
for loop ব্যবহার করবে।
if condition ব্যবহার করবে।
Math.max() এবং Math.min() ব্যবহার করা যাবে না।
reduce() ব্যবহার করা যাবে না।
Code clean হতে হবে।
Expected Output (format)
Highest Mark : ...
Lowest Mark : ...
Total Marks : ...
Average : ...
Pass Student : ...
